# Festival Aereo de Gijon 2026: где смотреть

Гид по авиашоу в Хихоне 24-26 июля 2026: карта с тремя точками наблюдения,
ориентировочная зона полётов (радиус 500 м от центра 43.546583, -5.651611)
и таблицы положения солнца для каждого борта и слота времени.

Страница: https://kryuchenko.github.io/gijon-airshow-2026/

Языки: испанский по умолчанию, `?hl=ru` — русский, `?hl=en` — английский
(переключатель есть и на самой странице).

## Точки наблюдения

1. **Лестница 10 (Escalera 10)** — середина пляжа Сан-Лоренсо. Лучшая точка
   для второй половины воскресного основного шоу: с ~13:05 солнце со спины
   до конца программы.
2. **Sombras de Luz** — скульптура на Муро между лестницами 18 и 19.
   Солнце со спины всё утро и до ~14:10 — оптимально для тренировок и
   первой половины воскресного шоу.
3. **Cerro de Santa Catalina** — холм у скульптуры Elogio del Horizonte.
   Лучшая точка для субботнего Sunset Air Show.

## Расчёт

Положение солнца считалось по алгоритму NOAA для каждой минуты слота
относительно азимута с точки наблюдения на центр полётов. Классификация:
солнце со спины (больше 135 градусов от направления взгляда), в глаза
(меньше 45 градусов), иначе сбоку. Если состояние меняется внутри слота,
в таблице показано то, что покрывает большую часть слота.

Оформление — дизайн-система [Astryx](https://astryx.atmeta.com/)
([facebook/astryx](https://github.com/facebook/astryx)): reset и базовые токены
подключены с CDN из `@astryxdesign/core`, поверх — тема `theme.css` в формате
`astryx theme build` с фирменными токенами astryx.atmeta.com; стили страницы
используют системные CSS-переменные (цвета, отступы, радиусы, тени).

Кэширование: GitHub Pages отдаёт файлы с max-age=600 (10 минут).
Внешние библиотеки прибиты к версиям на CDN, а `theme.css` подключается
с параметром `?v=<хэш содержимого>`. Перед коммитом запускайте
`./bump-version.sh` (или поставьте его pre-commit-хуком:
`cp bump-version.sh .git/hooks/... `, см. скрипт) — он обновит хэш
в `index.html`, и браузеры заберут свежий CSS сразу после обновления
страницы.

Карта построена на [Leaflet](https://leafletjs.com/) с тайлами OpenStreetMap.
Время выступлений ориентировочное, программа может меняться. Источники:
[gijon.es](https://www.gijon.es/es/eventos/festival-aereo-de-gijon),
[turismoasturias.es](https://www.turismoasturias.es).
